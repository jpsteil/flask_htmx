This Site
This site was built as a Single Page Application using Flask, htmx and bulma.
  • app.py - the Flask application
  • base.html - main template with links to resources (css, js, etc)
  • index.html - index page with the menu frame - is a child of base.html and contains a div with id="active_page" where htmx swaps in all content.
The data for the dropdowns and searches is built from a .csv file that is read when the app starts up. It is a simple dict() with the auto maker as the key pointing to a list of models sold by that make.
All code is available on github.
